# Gelby's Portfolio

Modern portfolio built with **Vite + React + Tailwind CSS + Framer Motion**.

## 🚀 Tech Stack
- React 18
- Vite 6
- Tailwind CSS 3
- Framer Motion 11
- React Icons

## 🛠️ Development

```bash
npm install
npm run dev
```

## 📦 Build & Deploy

```bash
npm run build    # build to /dist
npm run deploy   # deploy to GitHub Pages (gh-pages branch)
```

Automatic deployment via GitHub Actions on every push to `main`.
